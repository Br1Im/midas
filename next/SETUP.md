# Установка и запуск Midas Frontend

## Шаг 1: Установка зависимостей

```bash
cd next
npm install
```

## Шаг 2: Запуск dev сервера

```bash
npm run dev
```

Приложение будет доступно на http://localhost:3000

## Важно!

После изменения `tailwind.config.ts` необходимо перезапустить dev сервер:

1. Остановите сервер (Ctrl+C)
2. Запустите снова: `npm run dev`

## Дизайн

Приложение использует премиальную тёмную тему с золотыми акцентами:
- Тёмный фон (dark-900, dark-800)
- Золотые акценты (gold-400, gold-500)
- Градиенты для кнопок и элементов
- Анимации и hover эффекты
- Тени с золотым свечением

## Структура

- `/` - Дашборд
- `/leads` - Управление лидами
- `/campaigns` - Email кампании
- `/templates` - Шаблоны писем
- `/analytics` - Аналитика
- `/settings` - Настройки
