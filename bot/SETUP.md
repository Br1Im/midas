# Настройка Telegram бота Midas

## 1. Добавьте изображение приветствия

Поместите файл `hi.png` в папку `bot/`:

```
bot/
├── hi.png          # <- Ваше изображение приветствия
├── main.py
└── ...
```

## 2. Установка зависимостей

```bash
cd bot
pip install -r requirements.txt
```

## 3. Запуск бота

```bash
python main.py
```

## 4. Проверка

Откройте Telegram и найдите вашего бота, отправьте команду `/start`

Вы должны увидеть:
- ✅ Изображение hi.png
- ✅ Приветственное сообщение
- ✅ Список команд

## Команды бота

- `/start` - Приветствие с изображением
- `/help` - Подробная помощь
- `/parse` - Парсинг лидов из 2GIS
- `/leads` - Список лидов
- `/stats` - Статистика

## Примечание

Если файл `hi.png` не найден, бот отправит только текстовое приветствие без изображения.
