# 🚀 Быстрый запуск бота

## Шаг 1: Добавьте изображение

Поместите файл **hi.png** в папку `bot/`:

```
C:\Users\...\Midas\bot\hi.png
```

## Шаг 2: Установите зависимости

```bash
cd bot
pip install -r requirements.txt
```

## Шаг 3: Запустите бота

Дважды кликните на файл:
```
start_bot.bat
```

Или в командной строке:
```bash
python main.py
```

## Шаг 4: Проверьте бота

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте `/start`
4. Вы должны увидеть изображение hi.png и приветствие

## ✅ Готово!

Бот запущен и готов к работе!

## 📝 Примечание

- Токен бота уже настроен в `.env`
- Если hi.png не найден, бот отправит только текст
- Убедитесь что бэкенд запущен (http://localhost:8001)
