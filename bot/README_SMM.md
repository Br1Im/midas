# 🎨 Midas SMM Bot

Telegram бот для SMM агентства с AI-генерацией контента.

## ✨ Функции

### 📝 Генерация контента
- **Посты для соцсетей** - AI создаёт уникальные посты
- **Типы постов**: рекламные, обучающие, развлекательные, новостные, истории
- **Тоны**: профессиональный, дружелюбный, креативный, мотивирующий, юмористический
- **Платформы**: Instagram, VK, Telegram, Twitter, LinkedIn, YouTube

### 📅 Контент-план
- Автоматическая генерация плана на неделю
- Оптимальное время публикаций
- Разнообразие контента

### 💡 Идеи для постов
- 10+ готовых идей
- Актуальные тренды
- Проверенные форматы

### 📊 Аналитика
- Охваты и вовлечённость
- Лучшие посты
- Оптимальное время
- Рекомендации

## 🚀 Быстрый старт

### 1. Установка

```bash
cd bot
pip install -r requirements.txt
```

### 2. Настройка

Создайте `.env` файл:

```env
BOT_TOKEN=your_bot_token
OPENAI_API_KEY=your_openai_key  # Опционально
ANTHROPIC_API_KEY=your_claude_key  # Опционально
```

### 3. Запуск

```bash
python main.py
```

## 📱 Использование

### Основные команды

- `/start` - Главное меню
- `/help` - Справка
- `/generate` - Быстрая генерация поста
- `/plan` - Создать контент-план
- `/ideas` - Получить идеи
- `/stats` - Показать аналитику

## 🎯 Примеры использования

### Генерация поста

1. Нажмите "📝 Генерация контента"
2. Выберите "📝 Сгенерировать пост"
3. Выберите тип (например, "Обучающий")
4. Напишите тему (например, "SMM тренды 2024")
5. Выберите тон (например, "Профессиональный")
6. Выберите платформу (например, "Instagram")
7. Получите готовый пост!

### Контент-план

1. Нажмите "📅 Контент-план"
2. Получите план на неделю с 21 постом
3. Используйте как основу для своего календаря
