# Telegram Bot для Midas

Telegram бот для управления лидами и парсинга 2GIS.

## 🚀 Быстрый старт

### 1. Добавьте изображение

Поместите файл `hi.png` в папку `bot/`:

```
bot/
├── hi.png          # <- Ваше изображение приветствия
├── main.py
└── ...
```

### 2. Токен уже настроен

Токен бота уже добавлен в `.env` файл.

### 3. Установите зависимости

```bash
cd bot
pip install -r requirements.txt
```

### 4. Запустите бота

**Windows:**
```bash
start_bot.bat
```

**Linux/Mac:**
```bash
python main.py
```

## 📱 Использование

1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Вы увидите приветствие с изображением

## 🎯 Команды бота

- `/start` - Приветствие с изображением hi.png
- `/help` - Подробная помощь по командам
- `/parse` - Парсинг лидов из 2GIS
- `/leads` - Просмотр списка лидов
- `/stats` - Статистика по лидам

## ✨ Функции

- ✅ Приветствие с изображением
- ✅ Парсинг лидов через 2GIS
- ✅ Просмотр списка лидов
- ✅ Статистика
- ✅ Интеграция с API Midas

## 📋 Требования

- Python 3.11+
- Файл `hi.png` в папке bot/
- Запущенный бэкенд Midas (http://localhost:8001)

## 🔧 Разработка

Структура:
```
bot/
├── hi.png            # Изображение приветствия
├── main.py           # Основной файл бота
├── handlers/         # Обработчики команд
├── keyboards/        # Клавиатуры
├── utils/           # Утилиты
│   └── api.py       # API клиент
└── requirements.txt  # Зависимости
```

## ⚠️ Важно

Если файл `hi.png` не найден, бот отправит только текстовое приветствие.
